# Pastelería Java - Sistema de Gestión de Café y Postres

Sistema completo de gestión para cafetería que combina una API REST en **Node.js/Express** con un frontend en **React/Vite**
para manejar ventas, inventario y administración de personal.

> Nota: el código activo vive en la rama local `work`, pero debe publicarse como `main` en el remoto para que sea visible.

## Repositorio remoto
- URL prevista: `https://github.com/ALX1107/Proyecto_final_Java.git`.
